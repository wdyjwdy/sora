---
title: Safari 搜索后关闭键盘
category: Blog
---

- input 必须指定 enterkeyhint 属性，Safari 才会允许 blur 操作。

```html
<input type="text" enterkeyhint="search" />
```

- 通过 blur 关闭键盘。

```js
search.addEventListener("keydown", function (event) {
  if (event.code === "Enter") {
    this.blur(); // 关闭键盘
  }
});
```
