## Modes

| Command | Action       |
| ------- | ------------ |
| i       | Insert mode  |
| :       | Command mode |
| esc     | Normal mode  |

## Text

| Command | Action              |
| ------- | ------------------- |
| x       | Select current line |
| y       | Copy                |
| p       | Paste               |
| d       | Cut                 |
| u       | Undo                |
| U       | Redo                |
| R       | Replace             |

## File

| Command | Action        |
| ------- | ------------- |
| :w      | Save          |
| :q      | Quit          |
| :q!     | Force quit    |
| :wq     | Save and quit |

## Multiple Cursors

| Command | Action                  |
| ------- | ----------------------- |
| C       | Create new cursor below |
