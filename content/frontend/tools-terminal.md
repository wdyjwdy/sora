## Navigation

| Command | Action                  |
| ------- | ----------------------- |
| ls      | list non-hidden files   |
| ls -a   | list all files          |
| cd      | change directory        |
| pwd     | print working directory |

> `~`: home directory\
> `.`: current directory\
> `..`: parent directory

## File

| Command       | Action             |
| ------------- | ------------------ |
| rm hi.txt     | remove a file      |
| rm -r fruits/ | remove a directory |
| open hi.txt   | open a file        |
| open .        | open a directory   |

## Command

| Command                    | Action               |
| -------------------------- | -------------------- |
| bun install && bun run dev | sequential execution |

## Other

| Command        | Action                          |
| -------------- | ------------------------------- |
| Tab            | autocomplete                    |
| Up, Down       | cycle through previous commands |
| clear, cmd + k | clear the Terminal              |
| ctrl + c       | kill the server                 |
