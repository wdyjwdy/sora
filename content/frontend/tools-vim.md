## File

| Command   | Action    |
| --------- | --------- |
| vim a.txt | Open file |

## Modes

| Command | Action       |
| ------- | ------------ |
| i       | Insert mode  |
| :       | Command mode |
| esc     | Normal mode  |

## Text

| Command | Action         |
| ------- | -------------- |
| dd      | Cut            |
| p       | Paste          |
| :wq     | Write and Quit |
| :q      | Quit           |
