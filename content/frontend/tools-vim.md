## File

| Command   | Action                            |
| --------- | --------------------------------- |
| vim a.txt | Open file, default to normal mode |

## Modes

| Command | Action                              |
| ------- | ----------------------------------- |
| i       | Insert mode                         |
| :       | Command mode                        |
| esc     | Exit other modes, enter normal mode |

## Editing

| Command | Action                        |
| ------- | ----------------------------- |
| dd      | Normal mode, cut current line |
| p       | Normal mode, paste            |
| wq      | Command mode, save and exit   |
| q       | Command mode, exit            |
