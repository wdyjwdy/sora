## てしまう

| 书面                 | 口语               |
| -------------------- | ------------------ |
| 食べ*てしま*います   | 食べ*ちゃ*います   |
| 食べ*てしま*いました | 食べ*ちゃ*いました |
| 食べ*てしま*った     | 食べ*ちゃ*った     |
| 飲ん*でしま*います   | 飲ん*じゃ*います   |
| 飲ん*でしま*いました | 飲ん*じゃ*いました |
| 飲ん*でしま*った     | 飲ん*じゃ*った     |
