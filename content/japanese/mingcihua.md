## 名词する

去掉「する」。

```example
料理が好きだ。#喜欢做饭。
```

## ます形

去掉「ます」。省略部分助词「を・に・で」。

```example
帰りの切符。#回家的票。「帰る」
山の登りは楽しいです。#爬山很快乐。「山に登る」
私の趣味は魚釣りです。#我的兴趣是钓鱼。「魚を釣る」
この店は今日は休みです。#这家店今天休息。「休む」
```

## い形い

改为「い形さ」。

```example
高さは１０メートルです。#高十米。
```

## な形な

改为「な形さ」。

```example
東京の地下鉄の複雑さは有名です。#东京地铁的复杂度很出名。
```
