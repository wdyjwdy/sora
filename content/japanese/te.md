## 顺序动作

### 〔动て〕

1. 动作相继发生。

```example
早くうちに帰って夕食を食べましょう。#早点回家吃晚饭吧。
朝起きて、ご飯を食べて、それから学校を行く。#早上起床，吃饭，然后去学校。
```

## 并列动作

### 〔动て〕

1. 动作不分主次的排列。

```example
私は山へ行って、彼は川へ行った。#我去了山里，他去了河边。
私が歌て、彼女が踊る。#我唱歌，她跳舞。
この図書館は、広くて、新しい。#这个图书馆又大又新。
```

## 动作附加状态

### 〔动て〕

1. 表示在某种状态下的动作。

```example
着物を着て働く。#穿着和服工作。
窓を開けて寝る。#开着窗户睡觉。
スマホを持たないで出かける。#没带手机出门。
コーヒーに砂糖を入れて飲む。#加糖喝咖啡。
```

## 手段或方法

### 〔动て〕

```example
歩いて学校を行く。#走路去学校。
運動して痩せる。#通过运动减肥。
ナイフを使わないでパンを切る。#不使用刀切面包。
```

## 原因或理由

### 〔动て〕

后续可以接

- 心情：困る・寂しい
- 问候：ありがとう・すみません。
- 不能：行けない

后续不能接

- 希望：たい・ほしい・といい
- 意志：つもり・ようと思う
- 劝诱：ませんか・ましょう
- 命令：ください・いけない

```example
友達がなくて寂しい。#没有朋友，很寂寞。
美味しくて人気がある。#因为好吃所以很受欢迎。
仕事で忙しい。#因为工作所以很忙。
```

否定通常使用「なくて」。

```example
旅行に行けなくて、残念です。#没能去旅游，很遗憾。
```
